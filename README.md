# halo

ustack web framework extends egg

## Usage

```bash
npm i -S @ustack/halo
```

```file
# package.json
"egg": {
  "framework": "@ustack/halo"
}
```

## 插件

### 继承自egg的插件

+ onerror
+ session
+ i18n
+ watcher
+ multipart
+ security
+ development
+ logrotator
+ schedule
+ static
+ view
+ jsonp

## 中间件

### egg的中间件

+ meta
+ siteFile
+ notfound
+ bodyParser
+ overrideMethod
